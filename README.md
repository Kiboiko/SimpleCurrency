# simple_currency_cacher

Простой Ruby-гем для конвертации валют через публичный API с локальным кэшированием курсов. Включает Telegram-бота для удобного использоexport TELEGRAM_BOT_TOKEN=ваш_токен_бота
bundle exec botвания.

## 📌 Быстрый старт

```bash
bundle install
bundle exec rspec
```

## 🚀 Использование

### Как gem

```ruby
require 'simple_currency_cacher'

# Конвертирует 100 USD в RUB
result = SimpleCurrencyCacher.convert(100, from: 'USD', to: 'RUB')

# Конвертирует 100 EUR в GBP
result = SimpleCurrencyCacher.convert(100, from: 'EUR', to: 'GBP')
```

### Telegram-бот

1. Создайте бота в Telegram через @BotFather и получите токен.

2. Установите переменную окружения или передайте токен как аргумент:

```bash
export TELEGRAM_BOT_TOKEN=
bundle exec ruby bin/bot
```

Или:
```bash
bundle exec ruby bin/bot your_bot_token_here
```

#### Интерфейс бота

Бот имеет красивый и интуитивный интерфейс с кнопками:

- **Главное меню**: Приветственное сообщение с кнопками для конвертации и помощи.
- **Пошаговая конвертация**: 
  1. Введите сумму.
  2. Выберите валюту из списка популярных (USD, EUR, RUB и др.) или введите другую.
  3. Выберите валюту для конвертации.
- **Результат**: Показывает результат с курсом и кнопками для повторной конвертации или возврата в меню.

Команды:
- `/start` - главное меню
- `/convert` - начать конвертацию
- `/help` - справка

Поддерживаемые валюты: Любые валюты через API exchangerate-api.com.

## 🧠 Логика кэширования

Курс сохраняется в файл `~/.currency_cache.json`.
- Если с момента сохранения прошло меньше 24 часов — используется кэш.
- Если прошло больше 24 часов — скачиваются свежие курсы.

## 🚨 Обработка ошибок

- Если нет интернета и нет сохранённого кэша — вызывается `SimpleCurrencyCacher::NetworkError`.
- Если указана несуществующая валюта — вызывается `SimpleCurrencyCacher::CurrencyNotFoundError`.
